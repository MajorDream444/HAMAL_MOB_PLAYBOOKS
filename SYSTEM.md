# SYSTEM

## Core Model

Context → Agent → Skill → Tool → API → MCP → Plugin

## Definitions

### Context
The truth layer that defines how decisions are made.

### Agent
Execution layer that performs work.

### Skill
A documented method for solving a problem.

### Tool
An action layer (Airtable, Notion, Firecrawl, Telegram).

### API
Connection layer between tools and external systems.

### MCP
Standardized interface for tool usage.

### Plugin
Bundled system of skills + tools + config.

## Rule

No execution without context.
