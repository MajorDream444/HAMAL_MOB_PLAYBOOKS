# Rubrics

Status: Stage A foundation

This folder stores quality standards.

Rubrics judge whether an agent, skill, SOP, playbook, audit, or output is good enough to reuse.

## Rubric Should Answer

- What does good look like?
- What fails?
- What must be present?
- What should never happen?
- What requires human review?
- How is the output scored?

## Stage A Rule

Rubrics are required before autonomous execution.
