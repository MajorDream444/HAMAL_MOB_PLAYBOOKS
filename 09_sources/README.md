# Sources

Status: Stage A foundation

This folder stores source notes from transcripts, videos, documents, repos, screenshots, and external references.

Sources are not automatically doctrine.

They become doctrine only after review, extraction, and routing into the correct context, SOP, playbook, skill, rubric, or audit.

## Source Types

- transcript notes
- video notes
- repo reviews
- market signals
- tool docs
- screenshots
- client or user research notes
- founder voice notes

## Safety

Do not store secrets or sensitive client information here unless explicitly approved and sanitized.
