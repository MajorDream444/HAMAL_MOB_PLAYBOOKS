# SOPs

Status: Stage A foundation

This folder stores standard operating procedures for repeatable HAMAL work.

SOPs should be step-by-step and testable.

## Purpose

Use this folder for procedures such as:

- creating context files
- creating playbooks
- creating skills
- creating rubrics
- auditing repos
- auditing tools and MCPs
- preparing handoffs

## Stage A Rule

No autonomous execution.

No live tool calls.

No production integrations.
