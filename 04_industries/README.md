# Industries

Status: Stage A foundation

This folder stores industry-specific systems and operating models.

Use it when a playbook or MOB pattern becomes reusable for a market.

Examples:

- high-performance personal brands
- education
- wellness
- sports
- creator businesses
- local services
- cultural archives
- finance and ownership systems

Every industry file should identify:

- audience
- pain points
- reusable playbooks
- relevant skills
- tools or MCPs
- quality standards
- risk notes
